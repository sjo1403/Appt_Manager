Directions: 
	Step 1) Enter login credentials. If authenticated, user will be taken to the Main Screen.
	Step 2) View customer records and scheduled appointments in the Customers and Appointments TableViews, respectively. Scheduled appointments can be filtered by week, month, or all; select desired filter in 'View' section. Click 'Generate Report' to view a summary of all customer records and scheduled appointments.
	Step 3) Add, update, and delete customer records using the specified buttons in the Customer section.
	Step 4) Add, update, and delete appointments using the specified buttons in the Appointment section.

Custom Report: CUSTOMERS BY COUNTRY- All customer names will be organized by Country as determined by the information found in each customer record.
